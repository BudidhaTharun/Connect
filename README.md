## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/messagekaro.git
   cd messagekaro
   ```

2. **Install dependencies:**
   - Backend:
     ```bash
     cd backend
     npm install
     ```
   - Frontend:
     ```bash
     cd ../frontend
     npm install
     ```

3. **Set up `.env` files** for MongoDB and Cloudinary credentials.

4. **Start the app:**
   - Backend:
     ```bash
     npm start
     ```
   - Frontend (new terminal):
     ```bash
     npm start
     ```
